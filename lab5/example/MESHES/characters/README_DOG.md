# Модель собаки для проекта AGS

## Описание

Модель собаки (`E 45 Aircraft_obj.obj`) состоит из 80 вершин и включает следующие части:
- **Туловище** (основное тело)
- **Голова** (основная форма)
- **Морда** (нос/целевая зона)
- **4 Лапы** (передние и задние)
- **Хвост** (с простой геометрией)
- **2 Уша** (левое и правое)

## Структура OBJ файла

Файл использует тройные координаты (v/vt/vn):
- **v** - координаты вершин (x, y, z)
- **vt** - текстурные координаты (u, v)
- **vn** - нормали вершин (nx, ny, nz)
- **f** - грани/полигоны (ссылаются на вершины, текстуры и нормали)

## Генератор текстур (generate_dog_texture.py)

Скрипт позволяет создавать текстуры для модели собаки с различными окрасками.

### Требования
- Python 3.6+
- Встроенные библиотеки (не требуются внешние зависимости)
- PIL/Pillow (опционально, для PNG; без неё используется PPM)

### Использование

Базовое использование:
```bash
python3 generate_dog_texture.py --pattern brown
```

С опциями размера:
```bash
python3 generate_dog_texture.py --pattern spotted --width 1024 --height 1024
```

С пользовательским именем файла:
```bash
python3 generate_dog_texture.py --pattern dalmatian --output my_dog_texture.png
```

### Доступные окраски

1. **brown** (по умолчанию) - Коричневая собака с вариациями
2. **spotted** - Рыжая собака с коричневыми пятнами
3. **white** - Белая собака с серыми пятнами
4. **black** - Черная собака
5. **dalmatian** - Далматинец (белая с черными пятнами)

### Параметры скрипта

```
--pattern {brown|spotted|white|black|dalmatian}
    Выбирает тип окраски собаки (по умолчанию: brown)

--width WIDTH
    Ширина текстуры в пиксельях (по умолчанию: 512)

--height HEIGHT
    Высота текстуры в пиксельях (по умолчанию: 512)

--output FILENAME
    Имя выходного файла (по умолчанию: dog_texture.png)

--mtl FILENAME
    Имя файла материала MTL (по умолчанию: dog.mtl)
```

## Созданные файлы

После запуска скрипта создаются:
- **dog_texture.png** - Текстура коричневой собаки
- **dog_dalmatian.png** - Текстура далматинца
- **dog_spotted.png** - Текстура рыжей собаки с пятнами
- **dog.mtl** - Файл материала для использования с OBJ моделью

## Использование с проектом

1. **Подготовка модели:**
   - OBJ файл уже готов: `E 45 Aircraft_obj.obj`
   - Убедитесь, что файл находится в директории `lab5/example/MESHES/characters/`

2. **Генерация текстуры:**
   ```bash
   cd lab5/example/MESHES/characters
   python3 generate_dog_texture.py --pattern brown --width 1024 --height 1024
   ```

3. **Использование в коде (пример на C++):**
   ```cpp
   // Загружаем модель собаки
   Model dog("lab5/example/MESHES/characters/E 45 Aircraft_obj.obj");
   
   // Применяем текстуру
   dog.loadTexture("lab5/example/MESHES/characters/dog_texture.png");
   ```

## Информация о модели

- **Вершин:** 80
- **Текстурных координат:** 80
- **Нормалей:** 64
- **Граней:** 48
- **Масштаб:** Примерно 1.0 x 2.0 x 1.0 единицы
- **Центр:** Примерно в точке (0.5, 1.0, -0.5)

## Примечания

- Модель использует простую геометрию для учебных целей
- Все части соединены логически (голова над туловищем, лапы под туловищем и т.д.)
- Текстуры могут быть увеличены до 2048x2048 для лучшего качества
- Скрипт работает без внешних зависимостей благодаря встроенному Python движку

## Примеры создания разных текстур

```bash
# Коричневая собака (4K)
python3 generate_dog_texture.py --pattern brown --width 2048 --height 2048 --output dog_brown_4k.png

# Белая собака
python3 generate_dog_texture.py --pattern white --output dog_white.png

# Черная собака
python3 generate_dog_texture.py --pattern black --output dog_black.png

# Всем вместе
for pattern in brown spotted white black dalmatian; do
  python3 generate_dog_texture.py --pattern $pattern --output dog_${pattern}.png
done
```
